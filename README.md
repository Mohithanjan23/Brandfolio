# Mohithanjan — Unconventional Brand Strategist Portfolio

A high-impact, single-page portfolio designed for a multi-disciplinary freelancer. This site merges Neo-Brutalist aesthetics with functional "Figma-style" multiplayer interactions to create a unique, high-conversion digital experience.

## 🚀 The Philosophy

"Content is the next Commerce."
This portfolio is built for a hybrid strategist who operates across the entire creative and technical stack: Development, Design, Filmmaking, and Marketing. It rejects traditional agency "fluff" in favor of raw execution and 10x scaling results.

## ✨ Key Features

### 🖱️ Interactive Multiplayer Cursor

Context-Aware: The custom SVG cursor automatically adapts its label and color based on the element being hovered (e.g., "View Case," "Navigate," "Apply Now").

Visual Feedback: Built-in scaling effects and color transitions that match specific brand palettes for each case study.

### 🎨 Neo-Brutalist Design

High Contrast: Uses a bold palette of Acid Lime, Hazard Orange, Arterial Red, and Void Black.

Dynamic Backgrounds: An animated dot-grid canvas with floating, glowing glassmorphism orbs.

Agency Elements: Includes a scrolling ticker tape (marquee) and a rotating 360° SVG badge for immediate visual authority.

### 📊 "The Receipts" (Analytics Dashboard)

Data-First Proof: A brutalist dashboard mockup showcasing real growth metrics (55K+ impressions, 1,581% increase).

Responsive Scaling: Smart CSS logic prevents large metric numbers from breaking the layout on tablet and mobile views.

### 🏗️ CSS-Only Mockups

Custom-coded mobile app, 3D billboard, and video player mockups.

Zero dependency on external image files for UI elements, ensuring lightning-fast load times.

### ⏳ Scarcity Marketing Engine

Dynamic Availability: An "Availability Status" badge that automatically calculates and displays the upcoming month.

Selective Framing: Positions the freelancer as a high-demand partner with strict monthly bandwidth.

## 🛠️ Tech Stack

HTML5 / Vanilla JavaScript: Core structure and interactive logic.

Tailwind CSS: Utility-first styling and animations.

Google Fonts: Plus Jakarta Sans for a premium, modern feel.

SVG-Native Logic: Advanced text-on-path circular badges and custom iconography.

## 🔧 Customization

Change Projects: Update the "The Proof" section within index.html. Each card uses a data-hover-color attribute to control the interactive cursor color.

Update Analytics: Modify the h4 tags in the #analytics section to reflect your latest growth numbers.

Social Links: Links for LinkedIn, GitHub, and Instagram are located in the footer and final CTA section.

## 📱 Responsiveness

The interface is fully adaptive:

Mobile: Stacks elements vertically, hides secondary "fake" cursors to declutter, and adjusts typography scale.

Desktop: Enables the custom interactive cursor and full-width grid layouts.
